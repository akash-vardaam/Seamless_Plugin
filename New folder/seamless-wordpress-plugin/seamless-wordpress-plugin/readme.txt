== Description ==

Easily manage custom endpoints and shortcodes for your events and donations with client authentication.

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/seamless`.
2. Activate via 'Plugins' screen.
3. Set your client credentials in 'Seamless > Authentication'.

== Usage ==

* [seamless_event_list]
* [seamless_single_event id="1"]
* [seamless_single_donation id="1"]
